# ST\_Overflow {#reference_978556 .reference}

根据区域和区域中的降水量输出区域内代表水深度的像元值。

## 语法 {#section_f59_3dr_0bb .section}

``` {#codeblock_v0t_raj_3zm}
float8[] ST_Overflow(raster rast, Box box, float8 value);
```

## 参数 {#section_rta_bbn_9nu .section}

|参数名称|描述|
|----|--|
|rast|raster对象，目前仅支持1波段的DEM。|
|box|分析区域，世界坐标（与DEM一致）。|
|value|分析区域的水量（立方米）。|

## 描述 {#section_gzj_3kq_224 .section}

根据区域和区域中的降水量输出区域内代表水深度的像元值，raster对象目前仅支持1波段的DEM数据。
